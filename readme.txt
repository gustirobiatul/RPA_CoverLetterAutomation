This Automation is for making a Cover letter for those who still applying job. 
The format for the letter is on letter.docx and the content for the letter is on Source.xlsx
this work like this 
1. Read source.xlsx
2. get all variable that need to change (Date, Recruiter, Address, City, Position, Factory, Sender) 
3. Change the variable into word
4. Save as Company.doc 

feel free to change the format of letter or the content of the letter, but do not forget to change the word you want to change in format.docx too. 
Happy Automation!
